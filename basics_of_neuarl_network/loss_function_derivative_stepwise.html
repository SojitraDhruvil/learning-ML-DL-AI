Step by step derivation of MSE loss function derivative using power rule and chain rule

Shuru karo
Hamare paas yeh MSE Loss function hai. Hum iska A2 ke saath derivative nikalenge:
L = (1/m) × Σ (A2 − y)²

Jahan:
m = number of samples (4)
A2 = model prediction
y = actual answer
Yeh ek composite function hai — bahar ² hai, andar (A2−y) hai. Isliye Chain Rule lagega.
Step 1 — substitution
Pehle ek shortcut: u = (A2 − y) likhte hain, toh L simple dikhta hai:
u = A2 − y

L = (1/m) × u²

Ab hume chahiye: dL/dA2
Chain rule se: dL/dA2 = dL/du × du/dA2
Yeh substitution karna zaroori nahi, but step clear karne ke liye helpful hai.
Step 2 — Power Rule on L
dL/du nikalo — L = (1/m)×u², u ke saath differentiate karo:
L = (1/m) × u²

dL/du = (1/m) × 2u

Power Rule: d/du[u²] = 2u
(1/m) constant hai, woh bahar aa jaata hai

dL/du = 2u/m
Step 3 — Derivative of u
du/dA2 nikalo — u = A2 − y, A2 ke saath differentiate karo:
u = A2 − y

du/dA2 = d/dA2 [A2] − d/dA2 [y]

d/dA2 [A2] = 1 (khud ka derivative = 1)
d/dA2 [y] = 0 (y constant hai, A2 nahi)

du/dA2 = 1
Step 4 — Chain Rule combine karo
Ab dono parts ko multiply karo — yahi Chain Rule hai:
dL/dA2 = dL/du × du/dA2

dL/dA2 = (2u/m) × (1)

u ki jagah (A2 − y) wapas rakho:

dL/dA2 = 2(A2 − y) / m
Final answer — code mein
Yahi line code mein likhi thi! Ab puri tarah samajh aaya:
# Math:
dL/dA2 = 2 × (A2 − y) / m

# Python code (exactly same):
dL_dA2 = 2 * (A2 - y) / m

Agar A2=0.3, y=1.0, m=4:
dL_dA2 = 2 × (0.3 − 1.0) / 4
= 2 × (−0.7) / 4
= −0.35
Negative gradient → model ne kam predict kiya, toh A2 badhega next step mein.
