Java DSA Core Cheat Sheet

Essential predefined methods and structural conversions for fast coding 🚀

Arrays Methods
java.util.Arrays 
int[] arr = {5, 2, 9, 1, 3};

// Sort array (O(N log N))
Arrays.sort(arr);

// Print array
System.out.println(Arrays.toString(arr));

// Binary search (Array MUST be sorted first)
int index = Arrays.binarySearch(arr, 3);

// Fill array with a specific value
Arrays.fill(arr, 10);

// Compare two arrays
int[] arr2 = {10, 10, 10, 10, 10};
boolean isSame = Arrays.equals(arr, arr2);
Collections Methods
java.util.Collections 
List<Integer> list = new ArrayList<>(Arrays.asList(5,2,9,1,3));

// Sort and reverse
Collections.sort(list); 
Collections.reverse(list);

// Min and Max elements
int max = Collections.max(list);
int min = Collections.min(list);

// Find occurrences of a specific element
int count = Collections.frequency(list, 3);

// Randomly shuffle list elements
Collections.shuffle(list);
ArrayList (Dynamic Array)
O(1) Access 
ArrayList<Integer> al = new ArrayList<>();

al.add(10); // append to end
al.add(1, 20); // insert at index 1

al.remove(1); // remove by index
al.set(0, 100); // update value at index

int val = al.get(0); // access element
int size = al.size(); // get length
boolean hasVal = al.contains(100);
HashMap (Key-Value Pairs)
O(1) Lookup 
HashMap<Integer, String> map = new HashMap<>();

map.put(1, "A"); // insert/update
map.put(2, "B");

String val = map.get(1); // returns "A"
boolean hasKey = map.containsKey(2);
map.remove(2);

// Iterating Map
for (Map.Entry<Integer, String> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " -> " + entry.getValue());
}
Stack (LIFO)
Last In, First Out 
Stack<Integer> st = new Stack<>();

st.push(10); // add to top
st.push(20);

int top = st.pop(); // remove and return top (20)
int peek = st.peek(); // view top without removing (10)
boolean empty = st.isEmpty();
Queue (FIFO)
First In, First Out 
// Implemented using LinkedList
Queue<Integer> q = new LinkedList<>();

q.add(10); // add to rear
q.add(20);

int front = q.poll(); // remove and return front (10)
int peek = q.peek(); // view front without removing (20)
String Methods
Immutable 
String s = "hello world";

int len = s.length();
char c = s.charAt(1); // 'e'
String sub = s.substring(0, 5); // "hello"

boolean match = s.equals("hello world");
boolean has = s.contains("world");
int idx = s.indexOf('o'); // 4

// Split into array
String[] words = s.split(" "); 

// Sort a string
char[] chars = s.toCharArray();
Arrays.sort(chars);
String sortedStr = new String(chars);
Tip: If you are modifying a string inside a loop, always use StringBuilder to avoid Time Limit Exceeded (TLE) errors.

Crucial Type Conversions
Must Know for Assessments 
// 1. String ↔ Integer
int num = Integer.parseInt("123"); 
String backToStr = String.valueOf(num); 

// 2. String ↔ Char Array
char[] charArr = "hello".toCharArray(); 
String strAgain = new String(charArr); 

// 3. Object Array ↔ List
Integer[] arrObj = {1, 2, 3};
List<Integer> listFromArr = Arrays.asList(arrObj); 
Integer[] arrayFromList = listFromArr.toArray(new Integer[0]); 

// 4. Primitive Array ↔ List (Important ⚠️)
int[] primArr = {1, 2, 3};
// Cannot use Arrays.asList() for primitives! Must use a loop or streams.
List<Integer> primList = new ArrayList<>();
for (int val : primArr) primList.add(val);

// 5. List ↔ Set (Great for removing duplicates)
List<Integer> listDupes = Arrays.asList(1, 2, 2, 3);
Set<Integer> set = new HashSet<>(listDupes); 
List<Integer> listClean = new ArrayList<>(set); 

// 6. Map Keys/Values → List
List<Integer> keys = new ArrayList<>(map.keySet()); 
List<String> values = new ArrayList<>(map.values()); 

// 7. Binary String ↔ Integer
int decimal = Integer.parseInt("1010", 2); // returns 10
String binary = Integer.toBinaryString(10); // returns "1010"
