Total derivative interactive explainer — partial contributions from x and y

Socho ek pahaad hai — aap ek point par khade ho. Aap x direction mein bhi move kar sakte ho, y direction mein bhi. Total change = dono ka sum.
Ek variable:
df = (df/dx) × dx

Do variable (Total Derivative):
df = (∂f/∂x) × dx + (∂f/∂y) × dy

Teen variable:
df = (∂f/∂x)dx + (∂f/∂y)dy + (∂f/∂z)dz
∂ (partial) = baaki sab fixed, sirf ek se differentiate karo.
Total derivative = saare partial derivatives ka weighted sum.
Function: f(x, y) = x² + xy
Step 1: ∂f/∂x nikalo (y fixed rakho)
f = x² + xy
∂f/∂x = 2x + y

Step 2: ∂f/∂y nikalo (x fixed rakho)
f = x² + xy
∂f/∂y = x

Step 3: Total derivative likhho
df = (2x+y)·dx + (x)·dy
Ab x=2, y=3, dx=0.1, dy=0.2 rakho:
∂f/∂x = 2(2)+3 = 7
∂f/∂y = 2 = 2

df = 7×0.1 + 2×0.2
= 0.7 + 0.4
= 1.1 ← total change in f

Check: f(2,3)=4+6=10, f(2.1,3.2)=4.41+6.72=11.13
Actual change = 11.13−10 = 1.13 ≈ 1.1 ✓
Neural network mein Loss, hazaron weights W par depend karta hai. Total derivative hi batata hai sabka combined effect:
Loss function:
L = f(W1, W2, W3, ... Wn)

Total derivative (gradient):
dL = (∂L/∂W1)dW1 + (∂L/∂W2)dW2 + ... + (∂L/∂Wn)dWn

Gradient vector = saare partial derivatives:
∇L = [ ∂L/∂W1, ∂L/∂W2, ..., ∂L/∂Wn ]
Backpropagation actually total derivative hi calculate karta hai — har weight ka partial derivative nikalta hai Chain Rule se. Phir gradient descent se update karta hai:
W_new = W_old − lr × ∂L/∂W ← yahi total derivative ka use hai ML mein
